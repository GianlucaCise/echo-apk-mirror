# ⚠️ Important Disclaimer

This repository is an **automated mirror** of Echo APK builds.

## What This Repository Does

- Automatically downloads Echo APK from official nightly builds
- Creates versioned releases for easy access
- Provides checksums for verification

## What This Repository Does NOT Do

- ❌ Modify the APK in any way
- ❌ Claim ownership of Echo
- ❌ Provide support for Echo
  
## Official Echo Project

For official support, bug reports, and contributions:
- Visit: https://github.com/brahmkshatriya/echo
- Issues: Report to the official repository
- Development: Contribute to the official project

## Security Concerns

If you have security concerns about the APKs:
1. Verify checksums (provided in each release)
2. Download directly from nightly.link
3. Build from source at the official repository
