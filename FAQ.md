## General

### What is this repository?
An automated mirror of Echo nightly builds for easier access.

### Is this official?
No. The official repository is: https://github.com/brahmkshatriya/echo

### Is it safe?
Yes. APKs are downloaded directly from official nightly builds without modification.
Verify using the provided checksums.

## Installation

### How do I install?
1. Download the APK
2. Enable "Unknown Sources" in Android settings
3. Install the APK

### Why isn't it on Google Play?
This is a personal mirror. For official releases, check the original project.

### Can I install over the official version?
Yes, if the package name matches. Your data should be preserved.

## Updates

### How often are new versions released?
Automatically every Monday, or when manually triggered.

### How do I know when there's an update?
- Watch this repository
- Enable GitHub notifications
- Check the releases page

### Why wasn't X version released?
Only new versions are released. If Echo hasn't updated, no new release is created.

## Troubleshooting

### The APK won't install
- Check Android version compatibility (see release notes)
- Uninstall old version first
- Verify APK isn't corrupted (check checksums)

### How do I verify the APK?
```bash
md5sum echo-latest.apk
# Compare with MD5 in release notes
```
